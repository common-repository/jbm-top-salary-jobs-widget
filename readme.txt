== Description ==

"Job Board Manager - Top Salary Jobs Widget" adds a widget that displays the highest salary jobs.<br />
You can choose between minimum, maximum and fixed salary types.<br />
You can choose the number of jobs to display.<br />
You can also choose the salary currency.<br />
This plugin will only work with the <a href="https://wordpress.org/plugins/job-board-manager/">"Job Board Manager" plugin</a> by pickplugins.
<br />

== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Job Board Manager - Top Salary Jobs Widget</strong>" activate it.<br />
